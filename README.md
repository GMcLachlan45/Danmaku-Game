###On Windows

Create a new directory "/build"

_____

##On Linux

Create a new directory "/build".  
Then,   
`cd Danmaku-Game`  
Then,   
`cmake -S src -B build`

Then, run the following:
- `cd build`
- `make`

Then run    
`./danmaku`

And congratulations, you can run Danmaku-Game on Linux!
